# ProsjektHelseNorge → HelseFactor

**HelseFactor** is a health-focused web application that provides useful tools to help users understand and improve their health. The website includes calculators, quizzes, and other interactive features designed to make learning about health simple and engaging.

The project is built using **HTML, CSS, and JavaScript**, and also includes an **admin panel** which is not fully secured (since it is a school project).

**ProsjektHelseNorge** is the project name while **HelseFactor** is the name of the website.

https://helsefactor.netlify.app 

## Features

- **BMI Calculator** – Calculate your Body Mass Index to understand your weight category.
- **Calorie Calculator** – Estimate your daily calorie needs based on personal information.
- **Health Quiz** – Test your knowledge about health, nutrition, and lifestyle.
- **Admin Panel** – Manage and control parts of the website.
- **Responsive Design** – Works across desktop, tablet, and mobile devices.


## Built With

- **HTML5** – Website structure  
- **CSS3** – Styling and layout  
- **JavaScript** – Interactive functionality and calculations  

## Download / Clone

You can get a copy of the project in two ways.

### 1. Clone with Git (recommended)

If you have Git installed, run this command in your terminal:

```bash
git clone https://github.com/yourusername/prosjekthelsenorge.git
```

###  2.Download the project as a ZIP-file

Go to the GitHub repository page.

Click the Code button.

Select Download ZIP.

Extract the ZIP file on your computer.

Open the folder and run index.html in your browser.
